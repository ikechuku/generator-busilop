## Installation

1. Clone this repo

2. `cd` into the directory and install dependencies:

```sh
yarn
```

3. Symlink the generator so you can use it anywhere locally:

```sh
yarn link
```

4. Create a directory (where you want to scaffold your app) and start the generator by running:

```sh
yo busilop
```
